A modern, high-performance portfolio website built to showcase my projects and skills as a Computer Science student. Featuring abundant animations, a glassmorphism aesthetic, and dynamic GitHub integration.

## 🌟 Features

- **Dynamic GitHub Integration**: Automatically fetches and displays my latest "featured" projects using the GitHub API.
- **Modern UI/UX**: Built with a premium dark theme, glassmorphism effects, and smooth CSS animations.
- **Responsive Design**: Fully optimized for all devices, from mobile to desktop.
- **Interactive Elements**: Floating 3D visuals, hover effects, and smooth scrolling.

## 🛠️ Tech Stack

- **Frontend**: React.js (Vite)
- **Styling**: Vanilla CSS (Custom Properties & Animations)
- **Icons**: React Icons (Fa, Si)
- **Services**: GitHub REST API

## 🚀 Getting Started

To run this project locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/farhan4783/Portfolio.git
    cd Portfolio
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
